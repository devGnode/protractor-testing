# Protractor-testing

Protractor-tesing is a testing e2e framework

## prerequisites

* NodeJs 
* Npm
* Protractor
* Webdrvier-Manager

## SetUp


````shell script
$ npm install
````

* And then install the WebDriver, from this command

````shell script
$ npm Update
````


### Launch WebDriver

For launch the webDriver, you need to execv this command 

````shell script
$ npm start
````

### Launch Test


````shell script
npm test
````

- `target/cucumber-report.html`
